# 도메인 전문성 쌓기

> Data Science Career 101 시리즈 (9/10)


## 이 글에서 다룰 문제

*기술* 은 *복제* *가능*, *도메인* 은 *복리*.

## 전체 흐름
```mermaid
flowchart LR
    V[Vocab] --> M[Metric]
    M --> F[Field]
    F --> L[Loop]
```

## Before/After

**Before**: "*업종* 을 *모른* *채로* *대시보드* 만 *고친다*."

**After**: "*KPI* 와 *어휘* 로 *대화* *한다*."

## 5단계 도메인 학습

### 1단계 — 어휘집 만들기

```text
- 30개 단어
- 정의 + 예
```

### 2단계 — 핵심 지표 5개

```text
- 정의
- 계산식
- 사용 부서
```

### 3단계 — 현장 동행

```text
- 영업 또는 운영팀과 1일
- 메모와 질문
```

### 4단계 — 외부 학습

```text
- 업종 컨퍼런스 1건/분기
- 업계 뉴스 RSS
```

### 5단계 — 분기 회고

```text
- 새로운 단어 10개
- 새로운 지표 3개
```

## 이 코드에서 주목할 점

- *어휘* 가 *진입*.
- *지표* 가 *방향*.
- *현장* 이 *진실*.

## 자주 하는 실수 5가지

1. ***기술* 만 *판다*.**
2. ***어휘* 가 *없다*.**
3. ***지표 정의* 가 *모호*.**
4. ***현장* 을 *모른다*.**
5. ***외부 학습* 을 *건너* *뛴다*.**

## 실무에서는 이렇게 쓰입니다

핀테크, 헬스케어, 게임은 *도메인* 이 *답* 을 *바꿉니다*.

## 체크리스트

- [ ] *어휘* 30개.
- [ ] *KPI* 5개.
- [ ] *동행* 1회.
- [ ] *분기* 회고.
