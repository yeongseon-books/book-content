# DevOps 101 (5/10): Infrastructure as Code

мқҙ кёҖмқҖ DevOps 101 мӢңлҰ¬мҰҲмқҳ лӢӨм„Ҝ лІҲм§ё кёҖмһ…лӢҲлӢӨ.

## лЁјм Җ лҚҳм§ҖлҠ” м§Ҳл¬ё

- мҪҳмҶ”м—җм„ң нҒҙлҰӯн•ҙ л§Ңл“  мқён”„лқјлҠ” мҷң лӢӨлҘё нҷҳкІҪм—җм„ң мһ¬нҳ„н•ҳкё° м–ҙл Өмҡёк№Ңмҡ”?
- IaCлҠ” мҡҙмҳҒнҢҖмқҳ нҺёмқҳк°Җ м•„лӢҲлқј нҢҖ м „мІҙмқҳ ліҖкІҪ н’Ҳм§Ҳкіј м–ҙл–»кІҢ м—°кІ°лҗ к№Ңмҡ”?
- Terraformмқҳ кё°ліё нқҗлҰ„мқҖ planкіј applyлҘј мӨ‘мӢ¬мңјлЎң м–ҙл–»кІҢ мқҙн•ҙн•ҳл©ҙ мўӢмқ„к№Ңмҡ”?

## нҒ° к·ёлҰј

![DevOps 101 5мһҘ нқҗлҰ„ к°ңмҡ”](https://yeongseon-books.github.io/book-public-assets/assets/devops-101/05/05-01-diagram.ko.png)

*DevOps 101 5мһҘ нқҗлҰ„ к°ңмҡ”*

мқҙ к·ёлҰјмқҖ мҪ”л“ң м •мқҳ вҶ’ кі„нҡҚ кІҖнҶ  вҶ’ мӢӨм ң м Ғмҡ© вҶ’ мғҒнғң кё°лЎқмқҳ IaC нқҗлҰ„мқ„ ліҙм—¬мӨҚлӢҲлӢӨ. мҪ”л“ңнҷ”к°Җ лҗҳл©ҙ мқён”„лқј ліҖкІҪлҸ„ лҰ¬л·°мҷҖ мқҙл Ҙмқҳ лҢҖмғҒмқҙ лҗ©лӢҲлӢӨ.

> IaCмқҳ н•өмӢ¬мқҖ л¬ёлІ•мқҙ м•„лӢҲлқј, мқён”„лқјлҘј лӢЁмқј м¶ңмІҳ(SSOT)лЎң кҙҖлҰ¬н•ҳкі  ліҖкІҪмқ„ м¶”м Ғн•ҳлҠ” л¬ёнҷ”мһ…лӢҲлӢӨ.

## мҷң мӨ‘мҡ”н•ңк°Җ

мҪҳмҶ”лЎң л§Ңл“  мқён”„лқјлҠ” кё°м–өкіј нҷ”л©ҙ кё°лЎқм—җл§Ң лӮЁкё° мүҪмҠөлӢҲлӢӨ. к°ҷмқҖ кө¬м„ұмқ„ лӢӨлҘё лҰ¬м „мқҙлӮҳ лӢӨлҘё кі„м •м—җ лӢӨмӢң л§Ңл“Өл Өл©ҙ мӮ¬лһҢмқҙ лӢӨмӢң нҒҙлҰӯн•ҙм•ј н•ҳкі , к·ё кіјм •м—җм„ң мЎ°кёҲм”© лӢӨлҘё к°’мқҙ л“Өм–ҙк°Җл©ҙм„ң л“ңлҰ¬н”„нҠёк°Җ л°ңмғқн•©лӢҲлӢӨ.

IaCлҠ” мқҙ л¬ём ңлҘј кө¬мЎ°м ҒмңјлЎң мӨ„мһ…лӢҲлӢӨ. мқён”„лқјлҘј мҪ”л“ңлЎң м •мқҳн•ҳл©ҙ ліҖкІҪ мқҙл Ҙмқҙ лӮЁкі , PR лҰ¬л·°к°Җ к°ҖлҠҘн•ҙм§Җкі , к°ҷмқҖ м •мқҳлҘј м—¬лҹ¬ нҷҳкІҪм—җ л°ҳліө м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

> мҪ”л“ңлҠ” лӢЁмқј м¶ңмІҳмқҳ м§„мӢӨ(SSOT)мқҙм–ҙм•ј н•©лӢҲлӢӨ.

## н•ңлҲҲм—җ ліҙлҠ” к°ңл…җ

мқҙ нқҗлҰ„мқҳ мқҳлҜёлҠ” лӢЁмҲңн•©лӢҲлӢӨ. лЁјм Җ мҪ”л“ңлЎң м •мқҳн•ҳкі , ліҖкІҪ кІ°кіјлҘј лҲҲмңјлЎң нҷ•мқён•ң л’Ө, мӢӨм ң нҒҙлқјмҡ°л“ңм—җ л°ҳмҳҒн•ҳкі , к·ё кІ°кіјлҘј stateм—җ кё°лЎқн•©лӢҲлӢӨ.

## н•өмӢ¬ мҡ©м–ҙ

- **IaC**: мқён”„лқјлҘј мҪ”л“ңлЎң м •мқҳн•ҳкі  кҙҖлҰ¬н•ҳлҠ” л°©мӢқмһ…лӢҲлӢӨ.
- **Provider**: AWS, GCP, Azure к°ҷмқҖ нҒҙлқјмҡ°л“ң API м–ҙлҢ‘н„°мһ…лӢҲлӢӨ.
- **Resource**: мқёмҠӨн„ҙмҠӨ, лІ„нӮ· к°ҷмқҖ мғқм„ұ лӢЁмң„мһ…лӢҲлӢӨ.
- **State**: нҳ„мһ¬ мӢӨм ң мқён”„лқј мғҒнғңлҘј кё°лЎқн•ң нҢҢмқјмһ…лӢҲлӢӨ.
- **Module**: л°ҳліөн•ҙм„ң мһ¬мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мқён”„лқј л¬¶мқҢмһ…лӢҲлӢӨ.

мқҙ мҡ©м–ҙл“Өмқ„ мқҙн•ҙн•ҳл©ҙ Terraform л¬ёлІ•ліҙлӢӨ лЁјм Җ мҡҙмҳҒ мӣҗлҰ¬к°Җ ліҙмһ…лӢҲлӢӨ. нҠ№нһҲ stateлҘј мқҙн•ҙн•ҳм§Җ лӘ»н•ҳл©ҙ planкіј applyмқҳ мқҳлҜёлҸ„ л°ҳмҜӨл§Ң мқҙн•ҙн•ҳкІҢ лҗ©лӢҲлӢӨ.

## мқён”„лқј мҪ”л“ң лҸ„кө¬ л№„көҗ

IaC лҸ„кө¬лҘј м„ нғқн•  л•Ң к°ҖмһҘ нҳјлһҖмҠӨлҹ¬мҡҙ л¶Җл¶„мқҖ "Terraformмқ„ м“°л©ҙ лҗҳлӮҳ?"лқјлҠ” м§Ҳл¬ёмһ…лӢҲлӢӨ. м•„лһҳ н‘ңлҠ” л„Ө к°Җм§Җ лҢҖн‘ң лҸ„кө¬лҘј л№„көҗн•ҙ к°Ғ нҢҖмқҙ мһҗмӢ мқҳ нҷҳкІҪм—җ л§һлҠ” м„ нғқмқ„ н•  мҲҳ мһҲкІҢ н•©лӢҲлӢӨ.

| лҸ„кө¬ | м ‘к·јлІ• | м–ём–ҙ | мғҒнғң кҙҖлҰ¬ | м Ғн•© мғҒнҷ© |
|---|---|---|---|---|
| Terraform | м„ м–ёнҳ• | HCL (нҠ№нҷ” DSL) | мӣҗкІ© state нҢҢмқј | л©ҖнӢ° нҒҙлқјмҡ°л“ң, лӘЁл“Ҳ мғқнғңкі„ н’Қл¶Җ |
| Pulumi | м„ м–ёнҳ• + м Ҳм°Ёнҳ• | Python, TypeScript, Go л“ұ | нҒҙлқјмҡ°л“ң state лҳҗлҠ” мһҗмІҙ кҙҖлҰ¬ | кё°мЎҙ м–ём–ҙ мӮ¬мҡ© м„ нҳё, ліөмһЎн•ң лЎңм§Ғ н•„мҡ” |
| AWS CloudFormation | м„ м–ёнҳ• | YAML/JSON | AWS кҙҖлҰ¬ | AWS м „мҡ©, AWS нҶөн•© кёҠмқҢ |
| Ansible | м Ҳм°Ёнҳ• | YAML + Jinja2 | мғҒнғң кҙҖлҰ¬ м—ҶмқҢ (л©ұл“ұм„ұ кё°л°ҳ) | м„Өм • кҙҖлҰ¬ + мқён”„лқј н”„лЎңл№„м ҖлӢқ |

м ҲлҢҖ м •лӢөмқҖ м—ҶмҠөлӢҲлӢӨ. мӨ‘мҡ”н•ң кІғмқҖ нҢҖмқҳ нҒҙлқјмҡ°л“ң м „лһө, кё°мЎҙ кё°мҲ  мҠӨнғқ, нҳ‘м—… к·ңлӘЁм—җ л§һкІҢ кі лҘҙлҠ” кІғмһ…лӢҲлӢӨ. лҢҖл¶Җл¶„ нҢҖмқҖ Terraformмқ„ м„ нғқн•ҳм§Җл§Ң, Python нҢҖмқҙ PulumiлҘј м„ нҳён•ҳкұ°лӮҳ AWS м „мҡ© нҢҖмқҙ CloudFormationмқ„ м„ нҳён•ҳлҠ” кІғлҸ„ н•©лҠ¬м Ғмһ…лӢҲлӢӨ.

## нҢҢмқҙмҚ¬ н’ҖлЈЁлҜё мҳҲмӢң

Terraformмқҙ мЈјлҘҳмқҙм§Җл§Ң, Python к°ңл°ңмһҗм—җкІҢлҠ” Pulumiк°Җ лҚ” мһҗм—°мҠӨлҹ¬мҡё мҲҳ мһҲмҠөлӢҲлӢӨ. HCLмқ„ л°°мҡ°м§Җ м•Ҡкі  мқөмҲҷн•ң PythonмңјлЎң мқён”„лқјлҘј м •мқҳн•  мҲҳ мһҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

```python
# __main__.py
import pulumi
import pulumi_aws as aws

# S3 bucket for logs
log_bucket = aws.s3.Bucket(
    "my-app-logs",
    bucket="my-app-logs-prod",
    acl="private",
    tags={
        "Environment": "prod",
        "ManagedBy": "pulumi",
    },
    versioning=aws.s3.BucketVersioningArgs(
        enabled=True,
    ),
    server_side_encryption_configuration=aws.s3.BucketServerSideEncryptionConfigurationArgs(
        rule=aws.s3.BucketServerSideEncryptionConfigurationRuleArgs(
            apply_server_side_encryption_by_default=aws.s3.BucketServerSideEncryptionConfigurationRuleApplyServerSideEncryptionByDefaultArgs(
                sse_algorithm="AES256",
            ),
        ),
    ),
)

# Block public access
bucket_public_access_block = aws.s3.BucketPublicAccessBlock(
    "my-app-logs-public-access-block",
    bucket=log_bucket.id,
    block_public_acls=True,
    block_public_policy=True,
    ignore_public_acls=True,
    restrict_public_buckets=True,
)

# Export the bucket name
pulumi.export("bucket_name", log_bucket.bucket)
pulumi.export("bucket_arn", log_bucket.arn)
```

мқҙ мҪ”л“ңлҠ” S3 лІ„нӮ·мқ„ ліҙм•Ҳ лӘЁлІ” мӮ¬лЎҖм—җ л§һкІҢ мғқм„ұн•©лӢҲлӢӨ. лІ„м „лӢқ, м•”нҳёнҷ”, нҚјлё”лҰӯ м ‘к·ј м°ЁлӢЁмқҙ лӘЁл‘җ мҪ”л“ңм—җ лӘ…мӢңлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. Pythonмқҳ к°•л Ҙн•ң нғҖмһ… нһҢнҠёмҷҖ IDE м§Җмӣҗмқ„ к·ёлҢҖлЎң нҷңмҡ©н•  мҲҳ мһҲлӢӨлҠ” м җмқҙ Pulumiмқҳ мһҘм җмһ…лӢҲлӢӨ.

```bash
# Pulumi workflow
pulumi login       # state backend лЎңк·ёмқё
pulumi stack init prod
pulumi preview     # Terraform planм—җ н•ҙлӢ№
pulumi up          # Terraform applyм—җ н•ҙлӢ№
pulumi destroy     # лҰ¬мҶҢмҠӨ мӮӯм ң
```

## мқён”„лқј мҪ”л“ң н…ҢмҠӨнҠё

IaCлҘј мҪ”л“ңлЎң лӢӨлЈ¬лӢӨл©ҙ, мҪ”л“ңмІҳлҹј н…ҢмҠӨнҠён•ҙм•ј н•©лӢҲлӢӨ. мқён”„лқј н…ҢмҠӨнҠёлҠ” м„ё к°Җм§Җ л ҲлІЁлЎң лӮҳлҲ  ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

### 1л ҲлІЁ - plan н…ҢмҠӨнҠё (Syntax Check)

ліҖкІҪмқ„ мӢӨм ңлЎң м Ғмҡ©н•ҳкё° м „м—җ planмқҙ м„ұкіөн•ҳлҠ”м§Җ нҷ•мқён•ҳлҠ” кІғмқҙ к°ҖмһҘ кё°ліё н…ҢмҠӨнҠёмһ…лӢҲлӢӨ. л¬ёлІ• мҳӨлҘҳ, ліҖмҲҳ лҲ„лқҪ, provider м„Өм • л¬ём ңлҘј мЎ°кё°м—җ м°ҫмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

```bash
terraform validate    # л¬ёлІ• кІҖмҰқ
terraform fmt -check  # нҸ¬л§· кІҖмӮ¬
terraform plan        # ліҖкІҪ кі„нҡҚ нҷ•мқё
```

CI нҢҢмқҙн”„лқјмқём—җ мқҙ м„ё лӢЁкі„лҘј л„Јмңјл©ҙ PR лӢЁкі„м—җм„ң мқён”„лқј лІ„к·ёлҘј мһЎмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

### 2л ҲлІЁ - Policy as Code

л№„мҡ©, ліҙм•Ҳ, нғңк·ё м •мұ…мқ„ мһҗлҸҷ кІҖмҰқн•ҳл Өл©ҙ Open Policy Agent(OPA) лҳҗлҠ” Sentinel к°ҷмқҖ лҸ„кө¬лҘј н•Ёк»ҳ мӮ¬мҡ©н•©лӢҲлӢӨ.

```python
# OPA policy example (Rego)
package terraform.analysis

deny[msg] {
    resource := input.resource_changes[_]
    resource.type == "aws_s3_bucket"
    not resource.change.after.tags.Environment
    msg := sprintf("S3 bucket %s missing Environment tag", [resource.address])
}
```

мқҙ м •мұ…мқҖ лӘЁл“  S3 лІ„нӮ·м—җ Environment нғңк·ёк°Җ мһҲлҠ”м§Җ кІҖмӮ¬н•©лӢҲлӢӨ. нғңк·ёк°Җ л№ м§Җл©ҙ planмқҖ нҶөкіјн•ҳм§Җл§Ң policy лӢЁкі„м—җм„ң м°ЁлӢЁлҗ©лӢҲлӢӨ.

### 3л ҲлІЁ - нҶөн•© н…ҢмҠӨнҠё (Kitchen-Terraform)

мӢӨм ңлЎң лҰ¬мҶҢмҠӨлҘј мғқм„ұн•ҳкі , мғҒнғңлҘј нҷ•мқён•ҳкі , мӮӯм ңн•ҳлҠ” н…ҢмҠӨнҠёмһ…лӢҲлӢӨ. л№„мҡ©мқҙ л“Өкі  лҠҗлҰ¬м§Җл§Ң, мӨ‘мҡ”н•ң лӘЁл“ҲмқҖ мқҙ л ҲлІЁк№Ңм§Җ н…ҢмҠӨнҠён•ҙм•ј н•©лӢҲлӢӨ.

```ruby
# kitchen.yml (Kitchen-Terraform)
driver:
  name: terraform

provisioner:
  name: terraform

verifier:
  name: terraform
  systems:
    - name: default
      backend: ssh
      controls:
        - operating_system
```

мқҙ м„ё л ҲлІЁмқ„ лӘЁл‘җ к°–м¶”л©ҙ IaCлҠ” лӢЁмҲңн•ң мһҗлҸҷнҷ”к°Җ м•„лӢҲлқј н…ҢмҠӨнҠё к°ҖлҠҘн•ң мқён”„лқјк°Җ лҗ©лӢҲлӢӨ.
## м „нҷҳ м „нӣ„

**Before (console clicks)**

```text
- No record of *who* created what *when*
- Replicating to another region means *starting over*
- No change history
```

мқҙ кө¬мЎ°м—җм„ңлҠ” мҡҙмҳҒ м§ҖмӢқмқҙ л¬ём„ңмҷҖ мӮ¬лһҢ кё°м–өм—җ л¶„мӮ°лҗ©лӢҲлӢӨ. к°ҷмқҖ м„ңлІ„лҘј н•ҳлӮҳ лҚ” л§Ңл“Өкұ°лӮҳ нғңк·ё м •мұ…мқ„ нҶөмқјн•ҳлҠ” мқјлҸ„ л§ӨлІҲ мҶҗмңјлЎң лӢӨмӢң н•ҙм•ј н•©лӢҲлӢӨ.

**After (Terraform)**

```hcl
# main.tf
resource "aws_s3_bucket" "logs" {
  bucket = "my-app-logs"
  tags   = { Env = "prod" }
}
```

мҪ”л“ңнҷ”к°Җ лҗҳл©ҙ мқён”„лқј ліҖкІҪлҸ„ мҪ”л“ң лҰ¬л·°мҷҖ мқҙл Ҙ м¶”м Ғмқҳ лҢҖмғҒмқҙ лҗ©лӢҲлӢӨ. к·ё мҲңк°„л¶Җн„° мқён”„лқјлҠ” к°ңмқё л…ён•ҳмҡ°к°Җ м•„лӢҲлқј нҢҖ мһҗмӮ°мқҙ лҗ©лӢҲлӢӨ.

## н…ҢлқјнҸјмңјлЎң мӢңмһ‘н•ҳлҠ” лӢӨм„Ҝ лӢЁкі„

### 1лӢЁкі„ - Provider м •мқҳ

м–ҙл–Ө нҒҙлқјмҡ°л“ңмҷҖ м–ҙл–Ө лІ„м „мқ„ лҢҖмғҒмңјлЎң н• м§Җ лЁјм Җ лӘ…мӢңн•ҙм•ј н•©лӢҲлӢӨ. мқҙлҠ” лӢЁмҲңн•ң л¬ёлІ•мқҙ м•„лӢҲлқј мӢӨн–ү нҷҳкІҪмқ„ кі м •н•ҳлҠ” м„ м–ёмһ…лӢҲлӢӨ.

```hcl
terraform {
  required_providers {
    aws = { source = "hashicorp/aws", version = "~> 5.0" }
  }
}
provider "aws" { region = "us-east-1" }
```

### 2лӢЁкі„ - Resource мһ‘м„ұ

мқҙм ң мӢӨм ңлЎң л§Ңл“Өкі  мӢ¶мқҖ мқён”„лқјлҘј мҪ”л“ңлЎң н‘ңнҳ„н•©лӢҲлӢӨ. мӨ‘мҡ”н•ң м җмқҖ мӮ¬лһҢ м„ӨлӘ…мқҙ м•„лӢҲлқј м„ м–ёнҳ• мҪ”л“ңлЎң лӮЁкёҙлӢӨлҠ” мӮ¬мӢӨмһ…лӢҲлӢӨ.

```hcl
resource "aws_s3_bucket" "logs" {
  bucket = "my-app-logs-${var.env}"
}
```

### 3лӢЁкі„ - planмңјлЎң ліҖкІҪ нҷ•мқё

IaCм—җм„ң к°ҖмһҘ мӨ‘мҡ”н•ң мҠөкҙҖ мӨ‘ н•ҳлӮҳлҠ” apply м „м—җ planмқ„ мқҪлҠ” кІғмһ…лӢҲлӢӨ. лҲҲмңјлЎң ліҖкІҪ лӮҙмҡ©мқ„ нҷ•мқён•ҳм§Җ м•Ҡкі  л°”лЎң л°ҳмҳҒн•ҳл©ҙ мһҗлҸҷнҷ”к°Җ м•„лӢҲлқј мһҗлҸҷ мӮ¬кі к°Җ лҗ©лӢҲлӢӨ.

```bash
terraform init
terraform plan
# Plan: 1 to add, 0 to change, 0 to destroy.
```

### 4лӢЁкі„ - apply

applyлҠ” мӢӨм ң нҒҙлқјмҡ°л“ңлҘј ліҖкІҪн•ҳлҠ” мҲңк°„мһ…лӢҲлӢӨ. к·ёлһҳм„ң лҚ”лҚ”мҡұ plan лҰ¬л·°мҷҖ мӢӨн–ү к¶Ңн•ң, нҷҳкІҪ л¶„лҰ¬к°Җ н•Ёк»ҳ м„Өкі„лҸјм•ј н•©лӢҲлӢӨ.

```bash
terraform apply
# enters yes to actually create
```

### 5лӢЁкі„ - moduleлЎң мһ¬мӮ¬мҡ©

нҷҳкІҪл§ҲлӢӨ л№„мҠ·н•ң мқён”„лқјлҘј л°ҳліөн•ҙм•ј н•ңлӢӨл©ҙ лӘЁл“Ҳнҷ”к°Җ н•„мҡ”н•©лӢҲлӢӨ. мўӢмқҖ лӘЁл“ҲмқҖ нҢҖ н‘ңмӨҖмқ„ мҪ”л“ң мҲҳмӨҖм—җм„ң к°•м ңн•©лӢҲлӢӨ.

```hcl
module "vpc" {
  source = "terraform-aws-modules/vpc/aws"
  version = "5.0.0"
  cidr    = "10.0.0.0/16"
}
```

## мӣҗкІ© мғҒнғң нҢҢмқјкіј мһ кёҲмқҖ мҷң лЁјм Җ к°–м¶°м•ј н• к№Ң

Terraformмқ„ нҳјмһҗ м—°мҠөн•  л•ҢлҠ” лЎңм»¬ stateлҸ„ м–јн•Ҹ кҙңм°®м•„ ліҙмһ…лӢҲлӢӨ. н•ҳм§Җл§Ң нҢҖмқҙ л‘ҳл§Ң лҗҳм–ҙлҸ„ state м ҖмһҘ мң„м№ҳмҷҖ мһ кёҲмқҙ кі§ мҡҙмҳҒ мқҙмҠҲк°Җ лҗ©лӢҲлӢӨ. н•ң мӮ¬лһҢмқҖ `apply` мӨ‘мқёлҚ° лӢӨлҘё мӮ¬лһҢлҸ„ к°ҷмқҖ мӢңм җм—җ `plan`мқҙлӮҳ `apply`лҘј лҸҢлҰ¬л©ҙ, state м¶©лҸҢкіј мһҳлӘ»лҗң diffк°Җ л№ лҘҙкІҢ мҢ“мһ…лӢҲлӢӨ.

к·ёлһҳм„ң мӢӨл¬ҙм—җм„ңлҠ” лҰ¬мҶҢмҠӨлҘј лҠҳлҰ¬кё°ліҙлӢӨ лЁјм Җ мӣҗкІ© stateмҷҖ lockingмқ„ мһЎмҠөлӢҲлӢӨ. AWS мҳҲмӢңлқјл©ҙ S3 + DynamoDB мЎ°н•©мқҙ к°ҖмһҘ нқ”н•©лӢҲлӢӨ.

```hcl
terraform {
  backend "s3" {
    bucket         = "my-tf-state"
    key            = "network/prod.tfstate"
    region         = "us-east-1"
    dynamodb_table = "terraform-locks"
    encrypt        = true
  }
}
```

мқҙ кө¬м„ұмқҳ н•өмӢ¬мқҖ м ҖмһҘ мң„м№ҳліҙлӢӨ нҢҖ лҸҷмӢңм„ұ м ңм–ҙмһ…лӢҲлӢӨ. мһ кёҲмқҙ м—Ҷмңјл©ҙ plan кІ°кіјлҘј лҜҝкё° м–ҙл ӨмӣҢм§Җкі , кІ°көӯ IaCк°Җ мҳӨнһҲл Ө лҚ” мң„н—ҳн•ң мһҗлҸҷнҷ”к°Җ лҗ©лӢҲлӢӨ.

## л“ңлҰ¬н”„нҠёлҘј м°ҫлҠ” нҷ•мқё м Ҳм°Ё

IaCлҘј мҚЁлҸ„ мҪҳмҶ” мҲҳлҸҷ ліҖкІҪмқҙ мҷ„м „нһҲ мӮ¬лқјм§Җм§ҖлҠ” м•ҠмҠөлӢҲлӢӨ. к·ёлһҳм„ң мҡҙмҳҒнҢҖмқҖ planмқ„ лӢЁмҲң мғқм„ұ лӢЁкі„лЎң ліҙм§Җ м•Ҡкі , мӢӨм ң мғҒнғңмҷҖ мҪ”л“ңк°Җ м–ҙкёӢлӮ¬лҠ”м§Җ к°җмӢңн•ҳлҠ” л“ңлҰ¬н”„нҠё м җкІҖ м Ҳм°ЁлЎңлҸ„ мӮ¬мҡ©н•©лӢҲлӢӨ.

```bash
terraform plan -detailed-exitcode
# 0 = no changes
# 2 = drift or intentional change detected
# 1 = error
```

мқҙ лӘ…л №мқ„ PR кІҖмҰқмқҙлӮҳ м•јк°„ м җкІҖм—җ л„Јм–ҙ л‘җл©ҙ "лҲ„к°Җ мҪҳмҶ”м—җм„ң мҶҗлҢ”лҠ”м§Җ"лҘј лҠҰм§Җ м•ҠкІҢ л°ңкІ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ. IaCмқҳ к°Җм№ҳлҠ” м„ м–ёнҳ• л¬ёлІ•ліҙлӢӨ мқҙлҹ° л°ҳліө кІҖмҰқ лЈЁн”„м—җм„ң лҚ” нҒ¬кІҢ л“ңлҹ¬лӮ©лӢҲлӢӨ.

## мқҙ мҪ”л“ңм—җм„ң лЁјм Җ лҙҗм•ј н•  м җ

- apply м „м—җ planмқ„ ліҙкі  ліҖкІҪмқ„ мӢңк°Ғм ҒмңјлЎң кІҖнҶ н•ҙм•ј н•©лӢҲлӢӨ.
- stateлҠ” лЎңм»¬мқҙ м•„лӢҲлқј мӣҗкІ© л°ұм—”л“ңм—җ мһҲм–ҙм•ј н•©лӢҲлӢӨ.
- лӘЁл“ҲмқҖ нҷҳкІҪ м°ЁмқҙлҘј ліҖмҲҳлЎңл§Ң лӮЁкё°кІҢ н•ҙ мӨҚлӢҲлӢӨ.

IaCмқҳ мҡҙмҳҒ н’Ҳм§ҲмқҖ л¬ёлІ•мқ„ м–јл§ҲлӮҳ м•„лҠҗлғҗліҙлӢӨ, ліҖкІҪмқ„ м–јл§ҲлӮҳ м•Ҳм „н•ҳкІҢ кІҖнҶ н•ҳкі  кіөмң н•  мҲҳ мһҲлҠҗлғҗм—җм„ң к°ҲлҰҪлӢҲлӢӨ.

## мһҗмЈј н•ҳлҠ” мӢӨмҲҳ 5к°Җм§Җ

1. **stateлҘј лЎңм»¬м—җ л‘җлҠ” мӢӨмҲҳ**мһ…лӢҲлӢӨ. нҢҖ нҳ‘м—… м¶©лҸҢкіј мң мӢӨ мң„н—ҳмқҙ м»Өм§‘лӢҲлӢӨ.
2. **stateм—җ мӢңнҒ¬лҰҝмқ„ нҸүл¬ёмңјлЎң лӮЁкё°лҠ” мӢӨмҲҳ**мһ…лӢҲлӢӨ. л°ұм—”л“ң м•”нҳёнҷ”мҷҖ м ‘к·ј нҶөм ңк°Җ н•„мҲҳмһ…лӢҲлӢӨ.
3. **мҪҳмҶ”м—җм„ң мҲҳлҸҷ ліҖкІҪн•ҳлҠ” мӢӨмҲҳ**мһ…лӢҲлӢӨ. к·ё мҲңк°„ мҪ”л“ңмҷҖ мӢӨм ң мқён”„лқјк°Җ м–ҙкёӢлӮҳкё° мӢңмһ‘н•©лӢҲлӢӨ.
4. **applyлҘј мҲҳлҸҷмңјлЎңл§Ң лҸҢлҰ¬лҠ” мӢӨмҲҳ**мһ…лӢҲлӢӨ. мҳӨнғҖмҷҖ мӮ¬кі  к°ҖлҠҘм„ұмқҙ м»Өм§‘лӢҲлӢӨ.
5. **productionм—җ destroyлҘј м§Ғм ‘ м№ҳлҠ” мӢӨмҲҳ**мһ…лӢҲлӢӨ. нҷҳкІҪ л¶„лҰ¬мҷҖ мҠ№мқё кІҢмқҙнҠёк°Җ л°ҳл“ңмӢң н•„мҡ”н•©лӢҲлӢӨ.

## мӢӨл¬ҙм—җм„ңлҠ” мқҙл ҮкІҢ мқҙм–ҙм§‘лӢҲлӢӨ

м„ұмҲҷн•ң нҢҖмқҖ Terraform CloudлӮҳ AtlantisлЎң PR кё°л°ҳ plan/applyлҘј мһҗлҸҷнҷ”н•©лӢҲлӢӨ. мқён”„лқј ліҖкІҪлҸ„ м• н”ҢлҰ¬мјҖмқҙм…ҳ мҪ”л“ңмҷҖ к°ҷмқҖ мҲҳмӨҖмңјлЎң лҰ¬л·°н•ҳкі  мҠ№мқён•ҳлҠ” л¬ёнҷ”к°Җ мһҗлҰ¬ мһЎмҠөлӢҲлӢӨ.

мһ‘мқҖ нҢҖмқҙлқјл©ҙ лЁјм Җ мӣҗкІ© state, plan лҰ¬л·°, нҷҳкІҪлі„ ліҖмҲҳ л¶„лҰ¬ м„ё к°Җм§Җл¶Җн„° к°–м¶”лҠ” нҺёмқҙ мўӢмҠөлӢҲлӢӨ. к·ё м„ё к°Җм§Җк°Җ IaCмқҳ мһ¬нҳ„м„ұкіј м•Ҳм „м„ұмқ„ лҢҖл¶Җл¶„ кІ°м •н•©лӢҲлӢӨ.

## мӢңлӢҲм–ҙ м—”м§ҖлӢҲм–ҙлҠ” мқҙл ҮкІҢ лҙ…лӢҲлӢӨ

- мҪҳмҶ”мқҖ мқҪкё° м „мҡ©м—җ к°Җк№қкІҢ лӢӨлӨ„м•ј н•©лӢҲлӢӨ.
- stateлҠ” л°ұм—…кіј lockingмқҙ н•„мҲҳмһ…лӢҲлӢӨ.
- лӘЁл“ҲмқҖ нҢҖ н‘ңмӨҖмқ„ к°•м ңн•ҳлҠ” лҸ„кө¬мһ…лӢҲлӢӨ.
- PRм—җм„ң plan diffлҠ” н•өмӢ¬ лҰ¬л·° м •ліҙмһ…лӢҲлӢӨ.
- нғңк·ё м •мұ…мқҖ л№„мҡ©кіј мҶҢмң к¶Ң м¶”м Ғмқҳ кё°ліёмһ…лӢҲлӢӨ.

## мІҙнҒ¬лҰ¬мҠӨнҠё

- [ ] лӘЁл“  мқён”„лқјк°Җ мҪ”л“ңлЎң м •мқҳлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.
- [ ] stateк°Җ мӣҗкІ© л°ұм—”л“ңм—җ м ҖмһҘлҗ©лӢҲлӢӨ.
- [ ] PRл§ҲлӢӨ plan кІ°кіјлҘј нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.
- [ ] нғңк·ё м •мұ…мқҙ м Ғмҡ©лҗ©лӢҲлӢӨ.

## м—°мҠө л¬ём ң

1. TerraformмңјлЎң S3 лІ„нӮ· н•ҳлӮҳлҘј м§Ғм ‘ л§Ңл“Өм–ҙ ліҙм„ёмҡ”.
2. ліҖмҲҳлЎң нҷҳкІҪмқ„ лӮҳлҲ  dev/prodм—җ к°ҷмқҖ мҪ”л“ңлҘј м Ғмҡ©н•ҙ ліҙм„ёмҡ”.
3. S3 + DynamoDB мЎ°н•©мңјлЎң мӣҗкІ© stateлҘј кө¬м„ұн•ҙ ліҙм„ёмҡ”.

## м •лҰ¬ л°Ҹ лӢӨмқҢ лӢЁкі„

IaCлҠ” мһ¬нҳ„ к°ҖлҠҘн•ң мқён”„лқјлҘј л§Ңл“ңлҠ” л°©лІ•мһ…лӢҲлӢӨ. лӢӨмқҢ кёҖм—җм„ңлҠ” к°ҷмқҖ мһ¬нҳ„м„ұмқ„ м• н”ҢлҰ¬мјҖмқҙм…ҳ мӢӨн–ү нҷҳкІҪм—җ м ңкіөн•ҳлҠ” м»Ён…Ңмқҙл„ҲмҷҖ л№Ңл“ңлҘј лӢӨлЈ№лӢҲлӢӨ.

## мқён”„лқј мҪ”л“ңнҷ”лҘј нҢҖ мҡҙмҳҒ н‘ңмӨҖмңјлЎң м •м°©мӢңнӮӨлҠ” м„Өкі„

IaCлҠ” мқён”„лқјлҘј мҪ”л“ңлЎң "мһ‘м„ұ"н•ҳлҠ” лҚ°м„ң лҒқлӮҳм§Җ м•ҠмҠөлӢҲлӢӨ. мҪ”л“ң лҰ¬л·°, мғҒнғң кҙҖлҰ¬, м¶©лҸҢ л°©м§Җ, к¶Ңн•ң л¶„лҰ¬к№Ңм§Җ л¬¶м—¬м•ј мӢӨл¬ҙм—җм„ң м•Ҳм „н•ҳкІҢ мһ‘лҸҷн•©лӢҲлӢӨ. Terraformмқҳ л¬ёлІ•ліҙлӢӨ мҡҙмҳҒ к·ңм№ҷмқҙ лҚ” мӨ‘мҡ”н•©лӢҲлӢӨ.

### мқён”„лқј мҪ”л“ң лҸ„кө¬ л№„көҗн‘ң

| лҸ„кө¬ | к°•м җ | м•Ҫм җ | м Ғн•©н•ң нҢҖ |
| --- | --- | --- | --- |
| Terraform | л©ҖнӢ°нҒҙлқјмҡ°л“ң, мғқнғңкі„ м„ұмҲҷ | state кҙҖлҰ¬ н•„мҡ” | лҢҖл¶Җл¶„мқҳ лІ”мҡ© нҢҖ |
| CloudFormation | AWS л„ӨмқҙнӢ°лёҢ нҶөн•© | AWS мў…мҶҚм„ұ лҶ’мқҢ | AWS мӨ‘мӢ¬ мЎ°м§Ғ |
| Pulumi | мқјл°ҳ м–ём–ҙ мӮ¬мҡ© | лҹ°нғҖмһ„ мқҳмЎҙм„ұ мҰқк°Җ | мҪ”л“ң мӨ‘мӢ¬ н”Ңлһ«нҸј нҢҖ |
| Ansible | кө¬м„ұ кҙҖлҰ¬ к°•м җ | м„ м–ёнҳ• мқён”„лқј лӘЁлҚё н•ңкі„ | м„ңлІ„ кө¬м„ұ мһҗлҸҷнҷ” мӨ‘мӢ¬ |

### мҡҙмҳҒ кҙҖм җ мІҙнҒ¬н‘ң

### лӘЁл“Ҳнҷ” м „лһө

1. л„ӨнҠёмӣҢнҒ¬, ліҙм•Ҳ, м• н”ҢлҰ¬мјҖмқҙм…ҳ кі„мёө лӘЁл“Ҳмқ„ л¶„лҰ¬н•©лӢҲлӢӨ.
2. лӘЁл“Ҳ мһ…л Ҙ ліҖмҲҳлҠ” мөңмҶҢнҷ”н•ҳкі  м¶ңл ҘмқҖ лӘ…мӢңм ҒмңјлЎң кҙҖлҰ¬н•©лӢҲлӢӨ.
3. нғңк·ё м •мұ…, лЎңк№…, м•”нҳёнҷ” кё°ліёк°’мқ„ лӘЁл“Ҳ лӮҙл¶Җм—җм„ң к°•м ңн•©лӢҲлӢӨ.
4. лІ„м „ н•ҖningмңјлЎң мҳҲкё°м№ҳ м•ҠмқҖ м—…к·ёл Ҳмқҙл“ңлҘј л°©м§Җн•©лӢҲлӢӨ.

IaCмқҳ мөңмў… лӘ©н‘ңлҠ” нҒҙлҰӯ мһ‘м—… м ңкұ°к°Җ м•„лӢҲлқј мһ¬нҳ„м„ұкіј лҰ¬л·° к°ҖлҠҘм„ұ нҷ•ліҙмһ…лӢҲлӢӨ. нҢҖмқҙ мқён”„лқјлҘј "м„ӨлӘ…"мқҙ м•„лӢҲлқј "мҰқлӘ…"н•  мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ.

### Terraform мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨмҷҖ нҷҳкІҪ л¶„лҰ¬

нҷҳкІҪ л¶„лҰ¬лҘј ліҖмҲҳ нҢҢмқјлЎңл§Ң мІҳлҰ¬н•ҳл©ҙ мӢӨмҲҳлЎң prodм—җ applyн•ҳлҠ” мӮ¬кі к°Җ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ лҳҗлҠ” л””л үн„°лҰ¬ л¶„лҰ¬лЎң мӢӨн–ү м»Ён…ҚмҠӨнҠёлҘј лӘ…нҷ•нһҲ лӮҳлҲ„лҠ” нҺёмқҙ м•Ҳм „н•©лӢҲлӢӨ.

```bash
terraform workspace new dev
terraform workspace new prod
terraform workspace select prod
terraform plan -var-file=environments/prod.tfvars
```

м¶”к°ҖлЎң `prevent_destroy` к°ҷмқҖ м•Ҳм „мһҘм№ҳлҘј н•өмӢ¬ лҰ¬мҶҢмҠӨм—җ м Ғмҡ©н•ҳл©ҙ м№ҳлӘ…м Ғ мӮӯм ңлҘј мӮ¬м „м—җ л§үмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

```hcl
lifecycle {
  prevent_destroy = true
}
```


лҳҗн•ң лӘЁл“Ҳ н’Ҳм§Ҳмқ„ мң м§Җн•ҳл Өл©ҙ мҳҲм ң мҪ”л“ңмҷҖ мһ…л Ҙ ліҖмҲҳ л¬ём„ңлҘј н•Ёк»ҳ м ңкіөн•ҙм•ј н•©лӢҲлӢӨ. нҢҖмқҙ лӘЁл“Ҳмқ„ м •нҷ•нһҲ мӮ¬мҡ©н•ҳм§Җ лӘ»н•ҳл©ҙ н‘ңмӨҖнҷ” лҢҖмӢ  мҡ°нҡҢк°Җ лҠҳм–ҙлӮ©лӢҲлӢӨ. лӘЁл“Ҳ лҰҙлҰ¬мҠӨ л…ёнҠём—җ breaking changeлҘј лӘ…нҷ•нһҲ лӮЁкё°лҠ” мҠөкҙҖлҸ„ н•„мҲҳмһ…лӢҲлӢӨ.


### IaC ліҖкІҪ мҠ№мқё лӘЁлҚё

мқён”„лқј ліҖкІҪмқҖ мҳҒн–Ҙ лІ”мң„к°Җ нҒ¬лҜҖлЎң мөңмҶҢ 2мқё лҰ¬л·°мҷҖ нҷҳкІҪлі„ мҠ№мқё к·ңм№ҷмқ„ л‘җлҠ” нҺёмқҙ м•Ҳм „н•©лӢҲлӢӨ. devлҠ” мһҗлҸҷ apply, prodлҠ” мҲҳлҸҷ мҠ№мқёмІҳлҹј л¶„лҰ¬н•ҳл©ҙ мӢӨмҲҳ л№„мҡ©мқ„ мӨ„мқј мҲҳ мһҲмҠөлӢҲлӢӨ.


## мҡҙмҳҒ м•өм»Ө: л°°нҸ¬, мқён”„лқј, кҙҖмёЎм„ұ, лҢҖмқ‘мқ„ н•ң мһҘмңјлЎң м—°кІ°н•ҳкё°

м•һм„  м„№м…ҳм—җм„ң к°Ғ мЈјм ңлҘј л”°лЎң м„ӨлӘ…н–ҲлӢӨл©ҙ, мқҙ м„№м…ҳмқҖ мӢӨл¬ҙм—җм„ң н•ң лІҲм—җ м—°кІ°н•ҙ м“°лҠ” мөңмҶҢ кө¬м„ұ мҳҲмӢңлҘј м ңкіөн•©лӢҲлӢӨ. н•өмӢ¬мқҖ нҷ”л Өн•ң лҸ„кө¬ мЎ°н•©мқҙ м•„лӢҲлқј, к°ҷмқҖ кё°мӨҖмңјлЎң ліҖкІҪмқ„ нҶөкіјмӢңнӮӨкі  л¬ём ңлҘј лҗҳлҸҢлҰҙ мҲҳ мһҲлҠ”к°Җмһ…лӢҲлӢӨ.

### CI/CD нҢҢмқҙн”„лқјмқё кіөнҶө YAML

```yaml
name: delivery-flow
on:
  pull_request:
  push:
    branches: [main]

jobs:
  ci:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.12"
      - run: pip install -r requirements-dev.txt
      - run: ruff check .
      - run: pytest -q

  deploy-stage:
    needs: ci
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: ./scripts/deploy_stage.sh
      - run: ./scripts/smoke_test.sh https://stage.example.com

  deploy-prod-canary:
    needs: deploy-stage
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: ./scripts/deploy_prod.sh --strategy canary --percent 10
      - run: ./scripts/check_slo.sh --window 5m
      - run: ./scripts/promote_or_rollback.sh
```

мқҙ нқҗлҰ„мқҳ мӢӨм „ нҸ¬мқёнҠёлҠ” м„ё к°Җм§Җмһ…лӢҲлӢӨ. мІ«м§ё, CI нҶөкіј м „м—җлҠ” м–ҙл–Ө л°°нҸ¬лҸ„ мӢңмһ‘н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. л‘ҳм§ё, stage нҶөкіј нӣ„м—җл§Ң productionмңјлЎң мҠ№кІ©н•©лӢҲлӢӨ. м…Ӣм§ё, production мҠ№кІ©мқҖ canary кҙҖм°° нҶөкіјлҘј мЎ°кұҙмңјлЎң к°•м ңн•©лӢҲлӢӨ.

### лӘЁлӢҲн„°л§Ғ/м•ҢлҰј м„Өм • мҳҲмӢң

# monitoring/alerts.yml
groups:
  - name: api-slo
    rules:
      - alert: ApiHighErrorRate
        expr: rate(http_requests_total{service="api",status=~"5.."}[5m]) / rate(http_requests_total{service="api"}[5m]) > 0.01
        for: 5m
        labels:
          severity: page
        annotations:
          summary: "API 5xx л№„мңЁ 1% мҙҲкіј"

      - alert: ApiHighLatencyP95

м•ҢлҰјмқҖ л§Һмқҙ мҡёлҰ¬лҠ” кІғмқҙ лӘ©н‘ңк°Җ м•„лӢҷлӢҲлӢӨ. мҡҙмҳҒмһҗк°Җ мӢӨм ңлЎң н–үлҸҷн•  мҲҳ мһҲлҠ” мӢ нҳёл§Ң лӮЁкё°кі , лӘЁл“  page м•ҢлҰјм—җ runbook л§ҒнҒ¬лҘј л¶ҷм—¬ лҢҖмқ‘ мӢңмһ‘ мӢңк°„мқ„ мӨ„м—¬м•ј н•©лӢҲлӢӨ.

### лё”лЈЁк·ёлҰ°/м№ҙлӮҳлҰ¬ мҠ№кІ© м Ҳм°Ё мҳҲмӢң

лё”лЈЁк·ёлҰ°мқҖ мҰүмӢң м „нҷҳкіј мҰүмӢң лЎӨл°ұм—җ мң лҰ¬н•ҳкі , м№ҙлӮҳлҰ¬лҠ” мң„н—ҳмқ„ мһ‘кІҢ лӮҳлҲ  кІҖмҰқн•ҳлҠ” лҚ° мң лҰ¬н•©лӢҲлӢӨ. м„ңл№„мҠӨ нҠ№м„ұкіј нҢҖ м—ӯлҹүм—җ л”°лқј м „лһөмқ„ кі лҘҙлҗҳ, мҠ№кІ©/мІ мҲҳ лӘ…л №мқ„ л°ҳл“ңмӢң лҹ°л¶Ғкіј мһҗлҸҷнҷ” мҠӨнҒ¬лҰҪнҠёлЎң н•Ёк»ҳ мң м§Җн•ҙм•ј н•©лӢҲлӢӨ.

мҡҙмҳҒм—җм„ңлҠ” "мһҳ м•„лҠ” мӮ¬лһҢ"ліҙлӢӨ "к°ҷмқҖ мҲңм„ңлҘј л”°лҘҙлҠ” нҢҖ"мқҙ лҚ” л№ лҘҙкІҢ ліөкө¬н•©лӢҲлӢӨ. к·ёлһҳм„ң runbookмқҖ м„ӨлӘ… л¬ём„ңк°Җ м•„лӢҲлқј мӢӨн–ү л¬ём„ңм—¬м•ј н•ҳл©°, кІҪліҙм—җм„ң н•ң лІҲм—җ м—ҙ мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ.

## мІҳмқҢ м§Ҳл¬ёмңјлЎң лҸҢм•„к°Җкё°

- **мҪҳмҶ”м—җм„ң нҒҙлҰӯн•ҙ л§Ңл“  мқён”„лқјлҠ” мҷң лӢӨлҘё нҷҳкІҪм—җм„ң мһ¬нҳ„н•ҳкё° м–ҙл Өмҡёк№Ңмҡ”?**
  - ліёл¬ёмқҳ кё°мӨҖмқҖ Infrastructure as CodeлҘј н•ң лҚ©м–ҙлҰ¬ к°ңл…җмңјлЎң ліҙм§Җ м•Ҡкі  мһ…л Ҙ, мІҳлҰ¬, кІҖмҰқ, мҡҙмҳҒ мӢ нҳёк°Җ л§ҢлӮҳлҠ” кІҪкі„лЎң лӮҳлҲ„м–ҙ нҷ•мқён•ҳлҠ” кІғмһ…лӢҲлӢӨ.
- **IaCлҠ” мҡҙмҳҒнҢҖмқҳ нҺёмқҳк°Җ м•„лӢҲлқј нҢҖ м „мІҙмқҳ ліҖкІҪ н’Ҳм§Ҳкіј м–ҙл–»кІҢ м—°кІ°лҗ к№Ңмҡ”?**
  - мҳҲм ңмҷҖ к·ёлҰјм—җм„ңлҠ” м–ҙл–Ө к°’мқҙ л“Өм–ҙмҳӨкі , м–ҙлҠҗ лӢЁкі„м—җм„ң л°”лҖҢл©°, м–ҙл–Ө кё°мӨҖмңјлЎң нҶөкіј лҳҗлҠ” мӢӨнҢЁн•ҳлҠ”м§ҖлҘј лЁјм Җ нҷ•мқён•ҙм•ј н•©лӢҲлӢӨ.
- **Terraformмқҳ кё°ліё нқҗлҰ„мқҖ planкіј applyлҘј мӨ‘мӢ¬мңјлЎң м–ҙл–»кІҢ мқҙн•ҙн•ҳл©ҙ мўӢмқ„к№Ңмҡ”?**
  - мҡҙмҳҒм—җм„ңлҠ” мқҙ нҢҗлӢЁмқ„ мІҙнҒ¬лҰ¬мҠӨнҠё, лЎңк·ё, н…ҢмҠӨнҠёлЎң лӮЁкІЁ лӢӨмқҢ ліҖкІҪм—җм„ңлҸ„ к°ҷмқҖ мӢӨнҢЁк°Җ л°ҳліөлҗҳм§Җ м•ҠкІҢ л§үм•„м•ј н•©лӢҲлӢӨ.
