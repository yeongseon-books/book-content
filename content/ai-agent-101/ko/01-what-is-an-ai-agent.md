# AI Agent란 무엇인가?

> AI Agent 101 시리즈 (1/10)

ChatGPT를 사용해 본 사람이라면 모델이 질문에 답하는 방식은 익숙합니다. 하지만 AI Agent는 한 걸음 더 나아갑니다. 단순히 대답만 하는 것이 아니라, 도구를 호출하고, 결과를 확인하고, 다음 행동을 결정합니다. 이 차이가 작아 보이지만, 실제로는 LLM을 단순 대화 상대에서 자동화 가능한 작업 수행자로 바꾸는 핵심입니다.

Agent의 본질은 **observe → think → act → check** 루프입니다. 상황을 관찰하고(observe), 무엇을 해야 할지 판단하고(think), 실제로 도구를 실행하고(act), 결과를 확인합니다(check). 이 루프가 목표를 달성할 때까지 반복됩니다. ChatGPT는 한 번 답하고 끝나지만, Agent는 여러 번 행동하며 목표를 추구합니다.

이번 글에서는 AI Agent의 핵심 개념을 다룹니다. ChatGPT와 Agent의 차이, Agent의 구성 요소, 실제 사용 사례, 그리고 Agent가 필요한 상황과 필요하지 않은 상황을 구분합니다.

## 핵심 요약

- AI Agent는 ChatGPT처럼 한 번 답하고 끝나는 것이 아니라, 목표를 달성할 때까지 observe → think → act → check 루프를 반복합니다.
- Agent의 핵심은 도구 호출 능력과 결과 기반 의사 결정입니다.
- 단순 Q&A에는 ChatGPT로 충분하지만, 복잡한 작업 자동화에는 Agent가 필요합니다.
